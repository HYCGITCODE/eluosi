## 截图规范

### 格式要求
- **格式**: PNG（推荐）或 JPEG
- **分辨率**: 1920x1080 或更高
- **命名**: `render-deploy-01.png`, `render-deploy-02.png` ...

### 标注要求
- 使用红色方框或箭头标注关键按钮/字段
- 敏感信息（如 API Key、密码）需打码处理
- 保持浏览器地址栏可见（展示 URL）

### 存储位置
```
/docs/images/render-deploy/
├── render-deploy-01.png
├── render-deploy-02.png
└── ...
```

## 拍摄建议

### 工具推荐
- **macOS**: Shift+Cmd+4（区域截图）
- **Windows**: Win+Shift+S（截图工具）
- **Linux**: Flameshot、Shutter
- **浏览器扩展**: Nimbus Screenshot、Lightshot

### 注意事项
1. 使用无痕模式拍摄，避免个人信息泄露
2. 统一浏览器窗口大小（建议 1920x1080）
3. 保持截图风格一致（边框、标注颜色）
4. 关键操作步骤需连续拍摄，避免跳跃
