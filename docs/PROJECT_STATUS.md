## 部署说明

### Railway 部署步骤

1. **连接 GitHub 仓库**
   - 登录 Railway.app
   - New Project → Deploy from GitHub repo
   - 选择 `HYCGITCODE/ai-news-pulse`

2. **配置环境变量**
   ```
   NODE_ENV=production
   PORT=3000
   ```

3. **部署启动**
   - Railway 自动检测 `railway.json` 配置
   - 自动安装依赖并启动服务
   - 分配公网域名 (如 `ai-news-pulse-production.up.railway.app`)

4. **验证部署**
   - 访问 `/health` 端点确认服务正常
   - 检查新闻抓取功能
