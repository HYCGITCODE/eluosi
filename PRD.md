## 🏗️ 技术栈确认

### 核心技术栈（老大指令：P0+P1 全部交付）
| 组件 | 技术选型 | 理由 |
|------|----------|------|
| **GUI 框架** | **PyQt6** | 支持复杂 UI 交互（多视图切换、拖拽、搜索过滤），现代化界面 |
| **数据存储** | **SQLite** | 轻量级，单文件，无需配置，适合桌面应用 |
| **打包工具** | **PyInstaller** | 成熟稳定，支持 Windows 单文件打包，包含 PyQt6 依赖 |
| **编程语言** | **Python 3.8+** | 快速开发，跨平台，生态丰富 |

### 技术决策说明
1. **为什么选 PyQt6 而不是 Tkinter？**
   - **P0+P1 功能需要更强大的 UI 框架**：周视图/日视图、拖拽、搜索过滤等复杂交互
   - PyQt6 提供更现代的 UI 组件和更流畅的用户体验
   - 支持更复杂的布局和多视图切换
   - 拖拽操作在 PyQt6 中更成熟稳定
   - **技术升级**: 从 Tkinter 升级到 PyQt6 以支持全部 P0+P1 功能

2. **为什么选 SQLite？**
   - 单文件数据库，无需安装配置
   - 支持 SQL 查询，方便后续扩展
   - 性能足够桌面应用使用

3. **打包策略**
   - 使用 PyInstaller 生成单文件可执行程序
   - 包含 `--onefile --windowed` 参数
   - 目标：用户只需下载一个 `.exe` 文件
