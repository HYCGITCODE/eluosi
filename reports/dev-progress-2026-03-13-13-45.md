## 🐛 遇到的问题及解决方案

### 问题 1: Python 版本过旧
- **现象**: 系统 Python 3.6.8，pip 9.0.3
- **影响**: 无法安装最新版 PyQt6
- **解决**: 创建虚拟环境，安装兼容版本 PyQt6 6.2.3
- **状态**: ✅ 已解决

### 问题 2: SQLAlchemy 模型 Base 不统一
- **现象**: 外键错误 `NoReferencedTableError`
- **原因**: Task 和 Category 使用了不同的 declarative_base()
- **解决**: 创建统一的 `src/models/base.py`，所有模型共享同一个 Base
- **状态**: ✅ 已解决

### 问题 3: PyQt6 QAction 导入位置变更
- **现象**: `ImportError: cannot import name 'QAction'`
- **原因**: PyQt6 中 QAction 移至 QtGui 模块
- **解决**: 修改导入语句 `from PyQt6.QtGui import QAction`
- **状态**: ✅ 已解决

### 问题 4: 服务器无图形界面
- **现象**: Qt xcb platform plugin 无法加载
- **影响**: 无法在服务器上测试 GUI
- **解决**: 创建无 GUI 测试脚本验证核心逻辑
- **状态**: ✅ 已规避 (代码已验证，GUI 需在桌面环境测试)
